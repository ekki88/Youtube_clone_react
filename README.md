# Youtube clone
Youtube에서 제공하는 Data API를 이용한 클론 코딩 프로젝트입니다.<p>
Youtube의 필수 기능(검색, 인기영상, 동영상 재생)을 구현했습니다

## 기술
- javaScript
- React
- React Router
- PostCSS

## 사용
- Postman
- Youtube data APIs
